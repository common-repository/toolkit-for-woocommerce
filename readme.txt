Toolkit for WooCommerce is essential toolkit for WooCommerce

== Description ==

This plugin will help you:
* Delete all products with one click
* Change item name on paypal to order ID.

And there will be many more useful features that will be updated in the future.

== Screenshots ==

1. Delete all products with one click
2. Change item name on paypal to order ID

== Installation ==

1. In your admin panel, go to Plugins -> Add New.
2. Type `Toolkit for WooCommerce` in the search form and press the 'Enter' key on your keyboard.
3. Click on the `Activate` button to activate the plugin.
